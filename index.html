Trilha do Aprendizado

Para o meu site eu escolhi deixar claro os avanços que obtive durante o ano, com isso, ao passear pela barra de navegação vemos desde apenas o básico do HTML, paginas com pouco CSS até chegar no site que desenvolvi sobre curitiba...
